# Solid Todo

Experimental project for using the Solid framework. Simple todo app.

## Scripts

### Install

```bash
$ npm i
```

### Run the app

```bash
$ npm run dev
```

### Vitest developer test suite

```bash
$ npm test
```

### Mutation test suite

```bash
$ npm run test:mutation
```

### Playwright end-to-end test suite

```bash
$ npm run test:e2e
```
